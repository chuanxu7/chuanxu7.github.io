卫星和一劳永逸

发表于 2018-07-08 | | 阅读次数:

中学物理学万有引力的时候就推算过飞行器成为绕地球飞行的卫星的初始速度，即宇宙第一速度。当时就觉得这是个很神奇的概念，因为它是个分水岭：没有达到这个速度的飞行器，失去动力后都会掉下来；而达到这个速度的飞行器，即便没有了动力，也能成为地球的卫星，而永远不会掉下来，有点一劳永逸的味道。

一劳永逸在我们的核心价值观中是不被提倡的，相反提倡的是艰苦奋斗，终身奋斗，这并没有错。不过一劳永逸的现象确实在客观世界里是存在的，就像达到宇宙第一速度的卫星，永远不会掉下来那样，同时它还能利用太阳能帆板持续的获取能量。对于永动机形式的一劳永逸我们绝对不要幻想，但是成为永不掉下来的卫星（或者更严谨的说，叫很长一段时间不掉下来的卫星，因为卫星在减速后还是会掉下来）我们不要认为它是天方夜谭。

但是，要成为卫星的代价是巨大的。为了让卫星能达到宇宙第一速度，火箭应运而生。研发、制造、发射的过程都非常复杂而充满风险，只有经受住了这些层层考验，才能成功的将卫星送入预定轨道。

有时候想起来，卫星和个人作品一样，很有相似性。某人花了很大的力气完成了某优秀作品，它就能持续不断的从这部作品中获得收益，比如作家写书获得的版权收入、音乐人做音乐的版权收入、软件公司开发的软件、研究者获得的专利、投资者的股权收入等等。

所以，一劳永逸也可以是一个人生策略，但是必须得花足够的力气造出火箭，让自己达到宇宙第一速度。
