信息时代的生存工具

发表于 2018-07-02 | | 阅读次数:

信息时代来了，信息大爆炸！手机、电脑等各种电子产品都在完没了的吸取我们有限的注意力，在工作、出行、上课、睡觉前等各种情形下。

之前听了知乎上的讲座 -《如何建立自己的阅读操作系统》，里面的一段话点醒了处于幻觉中的我：

信息爆炸没有错，可是知识并没有爆炸，爆炸的是垃圾，真正的知识生成成本非常高，根本没那么容易产生，所以认真读是不会错过真正的知识，反而会让你获取更多有用的正确的知识。

是的，我经常在课堂上讲交通研究的基本概念：数据、信息、知识。但现在发现，虽然我讲了这么久，但至今为止，我都没有把它们真正的理解清楚，特别是对知识的理解。因为，在我的实践中并没有用这些概念发挥它们应有的价值。

在这个“垃圾”横行的信息时代中，我们如何让自己的大脑保持“干净”，让大脑中存放的都是真知，是这个时代中自如生存所必须解决的关键问题。

在一期罗辑思维的节目“我们到底该信谁”中提到了科学的气质：女神范、冷漠和傲慢，因为这样才能够保证知识积累的可靠。对于我们也是一样，要使得我们自己的知识体系建立的可靠，那么也应该具有和科学一样的气质。换句话说，我们应该给自己的大脑装上一层防护罩，在必要的时候开启，防止大脑受到侵害；在大脑吸收知识的时候，我们又应该为大脑装上一层过滤网，防止有害信息的污染。

防护罩

时间防护罩

时间防护罩是非常有必要的存在。我自己把最常用的几种分为工作时间防护罩，睡眠时间防护罩，健身时间防护罩，家庭时间防护罩……总之，你可为你所珍视的东西设置时间防护罩。时间防护罩其实也很好理解，就是在设定的时间内只做特定的事情，避免其他因素的打搅。

我曾经就为自己设置过工作时间。那时还在上研究生，由于长期受困于注意力被各种莫名其妙的事情吸走，我和自己约定周一-周六为工作时间，必须全身心的投入到工作中，周日为休息时间，必须从工作中脱离出来。这个约定很容易让我对一些事情做出判断，比如又新出了什么电影、美剧，只要是在工作时间，我就一律不看；工作时间也很少去刷微博和微信；娱乐时间得到了控制，且都放到了周日。正是由于有了这个简单的“防护”，我在工作日期间效率开始大幅度提升；而周日我也用在家里做饭这一方法来迫使自己从工作中彻底脱离出来。那段时间让我体验到了效率之美。

当时由于压力很大，我也为自己设定了健身时间防护罩，也就是每天晚上9：00，无论刮风下雨打雷闪电（顶着闪电去跑步），我都会出现在操场跑10圈（约4.5公里），这个活动我坚持了操作2个月。想起来，那段时间若没有跑步的陪伴，若跑步的时间被换成了上网，后果真不堪设想。

睡眠时间防护罩，我一直在尝试建立，但仍然很难避免被冲击。“新生大学”发表了早起的终极奥秘就两字（早睡）的文章，其实是在讨论如何建立更加牢靠的早起防护罩，附上文章的链接：

手机防护罩

手机防护罩是为了自己不被手机所干扰，这一点我已经有了突破。这得得益于一本书——《把时间当做朋友》我至今还保留着一本有着他的作者签名的版本。作者在日常生活中是这样对待手机的：

“我在生活中生生剔除的一个被动并行任务就是“接电话”。从好多年前开始，我就把自己的手机设定为“静音”状态，从不主动接电话——而是每隔一两个小时查看一下电话，有必要的就打回去。当我有条件给自己安排一间不受人打搅的“工作室”后，终于再一次大幅消减了“被动并行”造成的时间和精力的浪费。”

这真的是一个很棒的方法论，它其实就是为我们设置了一个手机提醒的防护罩，使我们免于受提醒的干扰。对于手机这件事上，还得理解得更深入些。手机是一种即时通讯工具，它原本要求我们对他做出即时的影响。还有一些延时通讯工具，例如邮件、短信等，你可以选择处理它们的时间。但是很多人分不清即时通讯和延时通讯，而且还有大量的骚扰电话存在的今天，打开我们的手机防护罩就很有必要了。此外，利用手机刷微信、刷微博这些都是些耗费注意力的无底洞，需要通过多种防护罩共同隔离。

互联网防护罩

现在的办公几乎都离不开电脑。但，电脑的功能太强大了，既可以用来办公，也可以用来娱乐。例如我现在就形成了一个不良习惯，打开浏览器不由自主的开始输入www.ifeng.com看新闻，这个过程已经进入了我的潜意识，自动完成，不需要任何思考。而且有些软件也会通过自动弹窗来吸引你的注意力。这方面我的经验就是：（1）用苹果OS的操作系统，没有那么多得意外干扰；（2）卸载掉一切多余的软件；（3）重要的是同时启动时间防护罩来抵挡它们的冲击。

过滤网

过滤网解决的是这样一个问题: 我们需要某些东西，但是在获取这些东西的同时会不可避免的获取到其他我们不想要的东西，因此我们需要一个过滤装置，保证我们只是获取我们想要的东西，把不好的东西挡在外面。来源过滤

选书的基本原则

这个方面我不用写太多的东西，因为前辈已经写了，我实在难以超越。就像相传李白到了黄鹤楼本欲赋诗，但是看到崔颢的诗已经在这里了，而且水平太高，故欲题又止。崔颢的诗道：

昔人已乘黄鹤去，此地空余黄鹤楼。
黄鹤一去不复返，白云千载空悠悠。
晴川历历沼阳树，芳草萋萋鹦鹉洲。
日暮乡关何处是? 烟波江上使人愁。

投人

先前听到风投的一个原则：“投人”，近期有听到一个理论：你的收入是你五个最近的朋友的平均值。信息的质量和人有很大的关联，通过人来过滤是一个长期靠谱的方法。这方面我也略有实践。比如：长期追踪李笑来的足迹，包括知乎live、新生大学以及他写得书，做过的演讲；近期订阅了不少的得到专栏，里面有古典、吴伯凡、李笑来、吴军等等各领域的大牛。

渠道和媒介

这个是从新生大学写作课中听到的：

（1）当别人看朋友圈的时候，你看好文章；
（2）当别人看免费内容自high的时候，你更多的读付费内容；
（3）当别人看好文章的时候，你看好书(精选的付费内容)；
（4）当别人看通俗杂志的时候（读者文摘，青年文摘—内含有大量虚构故事），你看精英杂志（读书）
（5）当别人好的中文书，你看好的英文书；
（6）别人看好书的时候，你看高质量的学术文献；
（7）当别人用百度的时候，你用google；
（8）当别人不做搜索的时候，你做Research；
（9）别人顶多搜索互联网的时候，你付费检索各大图书馆！

知识结构过滤

在朋友圈“养生文”大行其道的今天，通过知识结构来过滤信息也是生存利器。

英语和计算机是新时代的我给自己定下的必修课。对于英语方面的学习，我是“学习关键期理论”的受害者，这个理论目前已经被驳斥得体无完肤，就是说在小的时候没学好，长大了也学不好。无稽之谈，祸害无穷！计算机方面，我现在很后悔大学本科阶段没有把计算机的造诣持续提高，仅仅停留在把计算机课程（计算机原理、C++、数据库）考上90的水平，这一耽误就是好多年。后面我会就英语和计算机这两个点专门各写一篇文章。

除了此两个之外，还有一些知识结构的过滤网可以使用，如心理学、经济学、策略学、脑科学，这些都是对我们的思考有极其重要价值的知识，值得涉足和积累。

知识结构这个概念我听说得很早，在中学的时候我就听说了，但是现在才明白了它的意思。中学的时候，我们的知识结构几乎都是个二选一的问题，文科还是理科。由于高考指挥棒，我们根本没有改变自己知识结构的自由。而现在这个阶段，我可以说，我能自主的选择和优化自己的知识结构了！

要在信息时代生存下来，防护罩和过滤网都是不可缺失的工具吧。
